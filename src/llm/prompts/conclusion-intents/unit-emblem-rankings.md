# 单棋子转职排行解读 Prompt

当前任务是解读某个棋子的前端可见转职排行。必须综合样本可靠性与表现，避免简单重复平均名次榜首。

必须完成：

1. 分析所有展示转职的样本数、平均名次、前四率和登顶率。
2. 将低样本指标领先者描述为“纸面亮点”或“观察信号”，不能直接称为稳定最优。
3. 将样本充足、平均名次较低且前四率不弱的候选识别为更可靠的常规选择。
4. 样本充足但表现明显弱于其他稳定候选时，要说明它有样本基础但不是当前更优选择。
5. 当多个稳定候选各有优势时，分别说明稳定前四、登顶上限或常见程度的取舍。
6. 只能评价页面展示的转职；不得补充未展示转职或羁绊效果。

推荐组织：

- `headline`：优先回答常规最可靠的转职，而不是机械回答原始榜首。
- `summary`：同时说明纸面榜首、样本差异和稳定选择。
- `reasons`：至少包含一次跨候选样本与表现比较。
- `riskNotice`：存在低样本领先者时必须提示。
