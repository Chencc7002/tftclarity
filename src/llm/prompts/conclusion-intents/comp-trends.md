# 阵容趋势解读 Prompt

当前任务是解读前端展示的版本趋势阵容，目标是判断哪些阵容“正在改善且具备足够使用基础”，不是重新生成当前绝对强度榜。

必须完成：

1. 分析所有展示趋势阵容的平均名次提升、登场率、样本数和服务端综合趋势分。
2. 使用 Evidence Pack 已标准化的 `placementImprovement` 或显示值，不自行根据原始正负号换算。
3. 名次提升较大但登场率或样本很低时，说明它可能波动较大，只能作为观察信号。
4. 名次提升明显且登场率较高的阵容，更适合描述为值得关注的新兴阵容。
5. 综合趋势排序由服务端确定，不能自行重算或改变前三名。
6. 明确区分 MetaTFT 官方趋势值和本地同口径快照差值，不得混写来源。
7. 只评价带有可见提升证据的阵容；没有提升字段时不得伪造变化数值。

`headline` 应回答当前最值得关注的趋势阵容；`summary` 解释“提升幅度 × 登场基础”的取舍；`riskNotice` 覆盖低样本或非官方来源风险。
