# 单装备排行与核心装备解读 Prompt

当前任务是根据前端展示的单装备候选，判断哪些是核心装备、次选装备或情境装备。必须分析所有展示候选，不能只复述第一名。

若 `itemRankingContext.specialAveragePlacementOnly=true`，当前是神器或光明装备专用排行：低于 `outlierSampleFloor` 的极低样本离群项已经预先清洗；对剩余候选严格按平均名次从低到高解释既定排名。样本数只用于风险提示，不得参与重排，也不得改用普通装备的覆盖率、样本量或综合表现排序逻辑。

必须完成：

1. 联合比较装备频率或覆盖、样本数、平均名次、前四率和登顶率。
2. “核心装备”需要同时具备较高可见频率或覆盖、足够样本以及不弱的表现；不能只凭某个指标第一就称为核心。
3. 区分：
   - 核心装备：可见频率高、样本可靠且表现稳定；
   - 次选装备：表现或样本不错，但覆盖或综合表现略低；
   - 情境装备：特定指标亮眼但样本低、覆盖低或只适合特定搭配。
4. 对低样本指标领先者明确说明其观察性质，不得替代高样本稳定选择。
5. 对样本很多但平均名次或前四率明显较弱的候选，说明它常见但不是当前更优选择。
6. `commonPairings`、`copyCounts` 只能补充搭配和重复件信息，不能取代候选总体指标。

结论范围只能覆盖前端展示的单装备候选。不得宣称未展示装备一定更差，也不得从三件套页面的隐藏数据补充核心装备结论。
